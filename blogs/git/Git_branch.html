    Git创建远程分支

    阿张blog 2021/12/22 git

    # Git创建远程分支

    新建一个本地分支并切换到当前分支：

    git checkout -b test（分支名称）
    1

    查看一下现在的分支状态:

    git branch
    1

    （*）表示当前所在分支

    把新建的本地分支push到远程服务器，远程分支与本地分支同名

    git push origin test:test
    1

    切换本地分支

    git checkout main
    1

    删除本地分支

    git branch -d test // -D强制删除
    1

    删除远程分支

    git push origin --delete test
    1
